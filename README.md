### `CapCuMot`
Aplicatie Java in care in clasa principala este inclus un constructor fara parametrii in care se genereaza un desen. Am folosit clasele aplicatiei Shapes, una din aplicatiile din directorul Examples din BlueJ. Clasa are 5 variabile de stare: 3 variabile de tip Circle, doua de tip Triangle si trei variabile de tip Square.
